# circle
#### presented by esoracanvas
これはパーティゲームまたはほかのプロジェクトを補助するツールです。
スマートフォンまたはタブレットのみで、ゲームを友人たちと遊ぶことができます。
ルールブックファイル(.rlb)を編集することで、ゲームを開発することができます。
